OPTIONS DASHBOARD

What to click:

1. Double-click "START HERE.command"
2. Wait for setup and startup to finish
3. When the Terminal window says the dashboard is running, open:
   http://127.0.0.1:3000

Useful files:

- START HERE.command
  Starts everything. It can install missing prerequisites and project packages.

- OPEN DASHBOARD.command
  Opens the dashboard in the browser.

- STOP Dashboard.command
  Stops the backend and frontend.

Important:

- If macOS blocks the file, right-click it and choose Open.
- On a new Mac, macOS may ask for approval to install developer tools or Homebrew.
- The first run can take a while.
